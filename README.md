# BeanDo 隐私政策

## 引言

BeanDo（以下简称"我们"）非常重视用户隐私保护。本隐私政策描述了我们如何收集、使用、存储和分享您的个人信息，以及您享有哪些权利。请您仔细阅读本隐私政策，了解我们的隐私保护实践。

本隐私政策适用于全球用户，我们会根据不同地区的法律要求，为用户提供符合当地法律的隐私保护。

## 数据存储与隐私

### 1. 数据存储
- **本地存储**：您的所有提醒事项、标签和个性化设置都存储在您的设备本地，使用Core Data技术进行管理
- **iCloud备份**：如果您选择启用iCloud备份，您的数据将备份到您自己的iCloud账户，仅用于数据备份目的
- **开发者访问限制**：我们无法访问您的任何数据，包括提醒事项内容、标签信息和个性化设置

### 2. 数据收集说明
- **不收集个人数据**：我们不会收集、存储或分析您的任何个人数据
- **无设备信息收集**：我们不会收集您的设备型号、操作系统版本或设备标识符
- **无使用数据统计**：我们不会收集您的应用使用频率或功能使用情况
- **无崩溃日志上传**：我们不会自动上传崩溃日志或其他诊断信息

## 信息的使用

我们仅在您的设备本地使用您的数据，用于以下目的：

1. **提供核心功能**：
   - 存储和管理您的提醒事项
   - 发送提醒通知
   - 同步数据到系统日历

2. **个性化服务**：
   - 保存您的设置，提供一致的使用体验
   - 根据您的偏好显示应用界面

## 信息的共享

我们不会与任何第三方共享您的信息，原因如下：

1. **无数据收集**：我们不收集任何可共享的用户数据
2. **本地处理**：所有数据处理都在您的设备本地完成
3. **iCloud隐私**：如果您使用iCloud备份，数据仅备份到您的iCloud账户中，遵循Apple的iCloud隐私政策

我们不会向任何第三方出售、出租或共享您的个人信息，也不会将您的信息用于广告或营销目的

## 数据安全

我们采取了以下安全措施保护您的数据：

1. **数据加密**：在传输和存储过程中对数据进行加密
2. **访问控制**：限制对用户数据的访问权限
3. **安全审计**：定期进行安全审计和漏洞扫描
4. **备份机制**：定期备份用户数据，防止数据丢失

## 用户数据权利

根据不同地区的法律要求，您可能享有以下数据权利：

### 1. 访问权
您有权访问我们持有的关于您的个人数据（如有）。

### 2. 更正权
您有权更正不准确或不完整的个人数据（如有）。

### 3. 删除权
您有权请求删除您的个人数据（如有）。

### 4. 限制处理权
您有权请求限制对您个人数据的处理（如有）。

### 5. 数据可携带权
您有权请求以结构化、常用和机器可读的格式接收您的个人数据（如有）。

### 6. 反对权
您有权反对我们对您个人数据的处理（如有）。

### 7. 撤回同意权
如果您之前同意我们处理您的个人数据，您有权随时撤回该同意。

### 8. 投诉权
您有权向当地数据保护机构投诉我们对您个人数据的处理。

由于我们不收集任何个人数据，上述权利的实际行使可能受到限制。如果您有任何疑问，请联系我们。

## 权限使用

### 1. 通知权限
- **用途**：发送提醒通知，确保您及时收到重要提醒
- **使用场景**：当提醒事项到期时，发送推送通知
- **控制方式**：您可以在设备设置中随时关闭通知权限

### 2. 日历权限
- **用途**：将您的提醒事项同步到系统日历，方便您在日历中查看
- **使用场景**：创建、更新或删除与提醒事项对应的日历事件
- **控制方式**：您可以在设备设置中随时关闭日历权限

### 3. iCloud备份权限
- **用途**：备份您的提醒数据和部分APP的设置
- **使用场景**：自动备份到icloud中进行备份
- **控制方式**：您可以在应用设置中随时关闭iCloud备份功能

### 4. 麦克风权限
- **用途**：用于语音输入功能，方便您通过语音快速创建提醒
- **使用场景**：当您使用语音输入功能时，录制您的语音
- **控制方式**：您可以在设备设置中随时关闭麦克风权限

### 5. 语音识别权限
- **用途**：将您的语音转换为文本，用于创建提醒事项
- **使用场景**：当您使用语音输入功能时，识别您的语音内容
- **控制方式**：您可以在设备设置中随时关闭语音识别权限
